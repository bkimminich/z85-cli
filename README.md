Command line client for ZeroMQ Base-85 encoding

## Getting Started
Install the module with:

```
npm install -g z85-cli
```

## Documentation

### Encoding
```
z85 --encode [-e] <value>
```

### Decoding
```
z85 --decode [-d] <value>
```

## Specification

Please refer to [32/Z85 - ZeroMQ Base-85 Encoding Algorithm](http://rfc.zeromq.org/spec:32).
